# Databricks Certified Data Engineer Associate — Personal Repository

## Overview

This repository is my personal workspace for preparing for the **Databricks Certified Data Engineer Associate** certification. It is based on a Udemy course and connected to my Databricks environment. The goal is to practice, improve my skills, and build a reference for real-world data engineering tasks.

## Environment
**Databricks Free Edition**

## Contents

- `labs/` – Notebooks and exercises from the course  
- `pipelines/` – Examples of ETL and data pipelines  
- `incremental_processing/` – Practice with CDC and incremental loads  
- `governance/` – Notes and examples on security and data governance  

## Usage

1. Clone the repository:  
   ```bash
   git clone https://github.com/Rafavermar/DatabricksDemoRepo.git
   ```
2. Import notebooks into Databricks or run them locally with the appropriate Spark environment.

3. Adjust configurations (paths, cluster settings, credentials) as needed.
